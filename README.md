# Контрольная № 5

## Технологии индустриального программирования

---

**Реализованы:**

Express.js сервер - создан и запущен
Маршруты - реализованы GET, POST, PUT, DELETE

Работа с параметрами - req.params (для ID), req.query (фильтрация)

Обработка тела запроса - через express.json()

Пользовательский middleware - логгер запросов

Статические файлы - раздача через express.static()

Модульная структура - разделение на routes, controllers, middleware
